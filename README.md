# Auto Totem

Esse projeto é o sistema **Auto Totem**. Para configurar e rodar localmente, siga os passos abaixo:

## Como configurar:

1. **Clone o repositório**:

   ```bash
   git clone <URL_DO_REPOSITORIO>
Entre na pasta do projeto:

cd auto-totem
Renomeie o arquivo .env.example para .env:

mv .env.example .env
Instale as dependências com o Composer:

composer install
Abra o Dev Container: No VS Code, pressione Ctrl + Shift + P (ou Cmd + Shift + P no Mac) e digite Dev Containers: Open Folder in Container.

Isso deve configurar o ambiente. Se precisar ajustar algo, altere o arquivo .env.
